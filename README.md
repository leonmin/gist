## Install

```bash
$ npm install @leonmin/gist
```

## Usage

```bash
# print your gist and your can paste it from clipboard
$ gist get your_key
# set your gist to local
$ gist set your_key your_gist
# list all gists
$ gist list
# delete gist by gist key
$ gist delete your_key
# clear all gist
$ gist clear
```
